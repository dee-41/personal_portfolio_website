## 🌟 Overview

This is my personal portfolio website — a fully responsive, single-page application designed to present my academic background, technical skills, and project work to recruiters, mentors, and collaborators.

The design follows a **pastel blue-purple glassmorphism** aesthetic with animated backgrounds, smooth scroll interactions, and modern typography. It is built entirely in vanilla HTML, CSS, and JavaScript — no frameworks, no build tools — keeping it lightweight, fast, and easy to deploy anywhere.

---

## ✨ Features

- 🎨 **Glassmorphism UI** — Frosted-glass cards with backdrop blur and soft gradients
- 📱 **Fully Responsive** — Adapts seamlessly across desktop, tablet, and mobile
- 🌀 **Smooth Animations** — Scroll-triggered reveals, floating orbs, and hover micro-interactions
- 🖱️ **Cursor Glow Effect** — Subtle ambient glow that follows the user's cursor
- 🔢 **Animated Progress Bars** — Skill proficiency bars that animate on scroll into view
- 🗂️ **Project Showcase** — Clean cards for each project with tags, descriptions, and links
- 📬 **Contact Form UI** — Functional-looking contact form with success state feedback
- 🔗 **Social Links** — Quick access to LinkedIn, GitHub, and email
- ⚡ **Zero Dependencies** — Pure HTML/CSS/JS, no npm install required
- 🚀 **Deploy-Ready** — Works out of the box on GitHub Pages and Netlify
